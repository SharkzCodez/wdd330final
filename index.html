Welcome to the canvas showcase!

This site displays some of the features of the canvas API.

The canvas API is used to draw and generate 2d graphics through JavaScript, such as lines, shapes, text, and other things. While this won't be a guide, hopefully it will interest you into seeing what else you can do with this simple and fun API!

Starting off with a line, by using ctx.beginPath, ctx.moveTo, ctx.lineTo, and ctx.stroke, you can pick where to start and end your line before executing the function in JavaScript.

The square utilizes the ctx.fillRect as well as the ctx.clearRect, giving the shape an opening. By using ctx.fillStyle, you can choose the color of your fill.

This one functions the same as the last, only we use ctx.fillRect in two different places, as well as changing the opacity of the fill.

Using grad.addColorStop can give us a lovely gradient, just remember to fill in a shape.

Not only can the canvas API create straight lines, it can use bezier curves with ctx.bevierCurveTo.

Showing off some of the things you can do with the canvas API makes us happy, we hope the creativity to be found will bring you the same joy!
